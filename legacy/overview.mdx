## Deprecated Agents

### forest-express-sequelize

**Package:** `forest-express-sequelize` v9

**Status:** End-of-Support (December 2024)

**Migration Path:** Migrate to `@forestadmin/agent` v1 with `@forestadmin/datasource-sequelize`

**Migration Guide:** [See Migration Guide](/guides/migration/from-v1/overview)

**Legacy Documentation:** [Agent v1 docs](/legacy/agents-overview)

## Why Migrate?

### Agent v2 Improvements

**Better Performance:**
- Faster query execution
- Optimized data loading
- Reduced memory footprint

**New Features:**
- Native TypeScript support (Node.js)
- Better plugin system
- Improved customization API
- Cross-data source relationships
- Better error handling

**Better Developer Experience:**
- Cleaner API
- Better documentation
- Active development and support
- Regular updates and new features

**Security:**
- Active security patches
- Modern authentication
- Better permission system

### Migration Support

We provide:
- ✅ Detailed migration guides
- ✅ Migration tools and scripts
- ✅ Support during migration
- ✅ Side-by-side running (v1 + v2 in parallel)

**Start migration:** [Migration Guide](/guides/migration/from-v1/overview)

## Accessing Legacy Docs

### Agent v1 Documentation
- [Agent v1 Full Documentation](/legacy/agents-overview)
- Contains complete v1 reference for JavaScript and Ruby agents

### PHP Documentation
- The legacy PHP agent (`forestadmin-php`) is end-of-life. PHP is not part of the current generation of agents — for new projects, use the [Node.js or Ruby agent](/get-started/intro-to-forest-admin).
