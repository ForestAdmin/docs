  **Do not start new projects with this agent.** `forest_liana` (Ruby on Rails) v9 reached
  End-of-Support in December 2024. This section is preserved for teams maintaining existing
  deployments and as a reference during migration.
</Warning>

This section documents the legacy `forest_liana` gem for Ruby on Rails (v9).

It has been replaced by [`forest_admin_rails`](/reference/agent-api/ruby) — ActiveRecord and Mongoid support, actively maintained.
