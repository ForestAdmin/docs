<Warning>
  **Do not start new projects with these agents.** `forest-express-sequelize` and
  `forest-express-mongoose` v9 reached End-of-Support in December 2024. This section is
  preserved for teams maintaining existing deployments and as a reference during migration.
</Warning>

This section documents the legacy Node.js agents:

| Package | Datasource |
| --- | --- |
| `forest-express-sequelize` v9 | SQL (Sequelize) |
| `forest-express-mongoose` v9 | MongoDB (Mongoose) |

They have been replaced by [`@forestadmin/agent`](/reference/agent-api/nodejs) — TypeScript-first, multi-datasource, actively maintained.
