## Overview

**Package:** `forest-express-sequelize` v9

**Status:** End-of-Support (December 2024)

**Migration Path:** Migrate to [`@forestadmin/agent`](/product/integration/setup/nodejs) v1 with `@forestadmin/datasource-sequelize`

## Why migrate?

The new agent architecture provides:
- **Better Performance:** Faster query execution and optimized data loading
- **TypeScript Support:** Full TypeScript support out of the box
- **Better Plugin System:** Easier customization and extension
- **Active Support:** Regular updates, security patches, and new features
- **Modern API:** Cleaner and more intuitive API design
