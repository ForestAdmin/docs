The Forest Admin n8n node lets you integrate Forest Admin into your automation workflows. Trigger actions, browse records, and react to Forest Admin events directly from n8n.

<Note>This feature may not be available on all plans. Check the pricing page for more information, or contact us to enable it.</Note>

## What you can do

- Trigger custom actions on records from within an n8n workflow
- Query collections and use Forest Admin as a data source
- Use Forest Admin data as input to other steps in your workflows

## Setup

1. Install the Forest Admin node from the n8n node library.
2. Authenticate using your Forest Admin API key.
3. Configure your workflow by selecting the collection, action, or query you want to use.
