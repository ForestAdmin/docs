Forest is an agent you run in your own infrastructure. You define the data model, the business logic, and the actions in code; Forest renders the operations UI and enforces governance on top. Your data never leaves your network.

```bash Install
npm install @forestadmin/agent
```

```ts agent.ts
import { createAgent } from '@forestadmin/agent';

const agent = createAgent({ authSecret, envSecret })
  .addDataSource(/* SQL, Mongo, Stripe, … */)
  .customizeCollection('transactions', collection =>
    collection.addAction('Flag for review', {
      scope: 'Single',
      execute: async (context, result) => {
        // your logic, your audit trail
      },
    }),
  );

agent.mountOnStandalone().start();
```
