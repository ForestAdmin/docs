## Overview

The Security tab provides essential security features for project management.

## Available Security Features

The Security tab provides four key security features:

<CardGroup cols={2}>
  <Card title="IP Whitelisting" icon="network-wired" href="/get-started/control/security/ip-whitelisting">
    Control which IP addresses can access your back-office
  </Card>

  <Card title="Auto Logout" icon="clock" href="/get-started/control/security/auto-logout">
    Force user session logout after a few minutes of inactivity
  </Card>

  <Card title="Two-Factor Authentication (2FA)" icon="shield-halved" href="/get-started/control/authentication/2fa">
    Enforce a second authentication factor for specific environments
  </Card>

  <Card title="User Provisioning (SCIM)" icon="users" href="/get-started/control/authentication/scim-okta">
    Scale user management through Identity Providers
  </Card>
</CardGroup>
