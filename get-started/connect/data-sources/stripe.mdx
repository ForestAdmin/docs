title: "Stripe"
description: "Connect Stripe to Forest Admin"
---

<Note>Only available for Node.js.</Note>

The Stripe datasource connects your Stripe account to Forest Admin, allowing you to browse and manage your Stripe data (customers, payments, subscriptions, and more) directly from your back-office.

## Installation

```bash
npm install @forestadmin-experimental/datasource-stripe
```

## Basic usage

```javascript
import { createAgent } from '@forestadmin/agent';
import { createStripeDataSource } from '@forestadmin-experimental/datasource-stripe';

const agent = createAgent(options);

agent.addDataSource(
  createStripeDataSource({
    secretKey: process.env.STRIPE_SECRET_KEY
  })
);
```
